## Tech Stack

- **Frontend Framework:** React.js
- **State Management:** Redux Toolkit
- **Styling:** Tailwind CSS
- **Icons:** Heroicons
- **Internationalization:** i18next
- **HTTP Client:** Axios
- **Routing:** React Router
- **Form Handling:** React Hook Form
